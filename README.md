A full-stack analytics platform that simulates real SaaS product data (users, subscriptions, feature usage) and surfaces key growth metrics — MRR trend, monthly churn, and cohort retention — through an interactive dashboard, with an AI layer that generates grounded executive summaries from the live data.

---

## ✨ Features

- 📈 **Core SaaS Metrics** — MRR trend, monthly churn rate, and cohort retention, computed with boundary-artifact trimming for accuracy
- 🤖 **AI Executive Summaries** — Natural-language insights generated by Llama 3.3 70B (via Groq), grounded strictly in the underlying data — not hallucinated
- 🗄️ **Realistic Synthetic Dataset** — Generated users, subscriptions, and feature-usage data with an intentionally embedded activation → retention pattern, for genuine analytical discovery
- 🎨 **Interactive Dashboard** — KPI cards, tabbed views, and custom styling built with Streamlit
- ⚛️ **React Frontend** — A separate Vite + React interface layer for a more polished, production-style UI
- 🐘 **PostgreSQL Backend** — Structured schema via SQLAlchemy, with a clean ingest → clean → transform → load pipeline

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/shaikumar11/saas-analytics-project.git
cd saas-analytics-project
```

### 2. Backend setup
```bash
cd backend
python -m venv .venv
.venv\Scripts\activate       # Windows
pip install -r requirements.txt
copy env.example .env        # add your PostgreSQL & Groq API credentials
python main.py
```

### 3. Generate the dataset
```bash
cd ../data
python generate_data.py
```

### 4. Run the Streamlit dashboard
```bash
cd ../dashboard
streamlit run app.py
```

### 5. Run the React frontend (optional)
```bash
cd ../frontend
npm install
npm run dev
```
